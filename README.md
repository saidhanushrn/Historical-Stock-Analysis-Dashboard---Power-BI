# Historical-Stock-Analysis-Dashboard---Power-BI
This project analyzes historical stock data for Reliance Industries using Power BI to visualize trends, volatility, and key trading metrics. It demonstrates end-to-end data cleaning, feature engineering, analysis, and dashboard creation for actionable insights.
Features and Functionality:
Data Cleaning & Preparation: Raw stock data processed and cleaned using Power Query, including handling missing values and data type conversions.
Feature Engineering: Created calculated columns and measures such as 7-day moving average (MA_7), daily return percentages, and 7-day rolling volatility to quantify trends and risk.
Advanced Tooltips: Detailed information available on hover for all main visuals to enrich user interaction.
Summary Sheet: Executive insights and summary of key observations and interpretations of the data.
Interactive Dashboards: Multiple report pages presenting:
Price and volume trends with slicers for year and trading activity.
Daily returns with conditional formatting for gains and losses.
Moving average overlay to smooth price fluctuations.
Volatility monitoring to assess market risk.
Highlight cards for key dates such as highest/lowest price and returns.

Technologies Used:
Power BI Desktop (DAX & Power Query)
CSV Data Files for stock price and volume
Data Visualization and Business Intelligence Best Practices

How to Use:
Download the Historical_Stock_Analysis.pbix Power BI file from this repository.
Open it in Power BI Desktop application.
Navigate through the various dashboard pages.
Use slicers to filter by year or specific return categories.
Hover over visuals for additional details via tooltips.
Read the summary sheet for critical insights.

Key Insights:
Reliance stock showed a strong upward trajectory in 2025 supported by positive investor sentiment.
Volatility spikes aligned with significant market events and were followed by relative stabilization.
Moving averages delineated clear trends smoothing daily noise.
Highest and lowest price/return dates are clearly highlighted for quick reference.
The combination of trend and risk indicators provides a balanced view to guide investment decisions.
